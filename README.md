BrainMovie: Système de Recommandation de Films

## Description du Projet
Ce projet PFE = vise à développer un système de recommandation de films interactif et personnalisé. L'application, construite avec Streamlit, utilise des techniques de filtrage basé sur le contenu pour suggérer des films aux utilisateurs en fonction de leurs préférences.

## Problématique
Le volume croissant de films disponibles sur les plateformes de streaming rend le choix difficile pour les utilisateurs. Un système de recommandation efficace est donc essentiel pour améliorer l'expérience utilisateur en proposant des films pertinents.

## Objectif Principal
Développer un système de recommandation de films basé sur des techniques de machine learning (filtrage basé sur le contenu) pour offrir des suggestions personnalisées à partir des caractéristiques des films.

## Technologies Utilisées
*   **Python**: Langage de programmation principal.
*   **Streamlit**: Pour la création de l'interface utilisateur web interactive.
*   **Pandas**: Pour la manipulation et l'analyse des données.
*   **Scikit-learn**: Pour l'implémentation des algorithmes de machine learning (notamment `TfidfVectorizer` et `linear_kernel`).
*   **Docker**: Pour la conteneurisation de l'application, facilitant le déploiement.
*   **Dataset MovieLens**: Ensemble de données de films et de notations utilisé pour entraîner et tester le système de recommandation.
